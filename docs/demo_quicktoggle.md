# CopyPathFinder Quick Toggle 功能演示

## 🎯 功能演示步骤

### 1. 启动应用
```bash
# 构建并运行应用
swift build
.build/x86_64-apple-macosx/debug/CopyPathFinder
```

### 2. 打开设置
- 右键点击状态栏中的 CopyPathFinder 图标
- 选择"设置"选项

### 3. 配置 Quick Toggle
- 切换到"快速切换"标签页
- 勾选"启用快速切换"
- 从下拉菜单选择操作类型：
  - 拷贝路径
  - 拷贝文件名  
  - 在终端中打开

### 4. 测试功能
- **左键点击**状态栏图标：直接执行选定的操作
- **右键点击**状态栏图标：显示传统菜单
- **Ctrl+点击**状态栏图标：显示传统菜单

### 5. 验证效果
- 选择文件/文件夹
- 左键点击状态栏图标
- 确认执行了相应的操作
