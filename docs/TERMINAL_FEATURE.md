# 终端打开功能

CopyPathFinder 现在支持在终端中打开选中的文件夹或文件。

## 新增功能

- **菜单项**: 在状态栏菜单中添加了 "💻 Open in Terminal" 选项
- **快捷操作**: 点击菜单项即可在终端中打开当前选中的路径
- **智能切换**: 自动切换到选中的目录，如果选择的是文件则切换到文件所在目录

## 使用方法

1. 在 Finder 中选择一个文件夹或文件
2. 点击状态栏中的 CopyPathFinder 图标
3. 在弹出菜单中选择 "💻 Open in Terminal"
4. 终端将自动打开并切换到对应路径

## 技术实现

- 使用 AppleScript 与 Terminal 应用程序交互
- 自动处理路径中的特殊字符（如单引号）
- 兼容 macOS 10.15 及以上版本
- 提供操作成功/失败的通知反馈

## 兼容性

- ✅ macOS 10.15+ (本地编译)
- ✅ macOS 11/12 (GitHub CI)
- ✅ Terminal.app (macOS 默认终端)
- ✅ 支持包含特殊字符的路径
